# Password-Generator-Project
It generates strong and unique passwords.

User Input: Prompt the user to specify the desired length of the password.It must be greater than or equal to  12 characters.

Ensure Uniqueness: User can select if she wants a unique password with no repetitions or not.

Generate Password: Using logic and a combination of random characters from the types selected by the user to generate a password of the specified length. Atleast 3 character types must be chosen for a strong password.

Display the Password: The password generate is then printed on the screen which the user can copy using the"Copy to Clipboard" button.
